# Customer Service Documentation

## Endpoints

- `/api/customers/` (GET): Danh sách khách hàng
- `/api/customers/register/` (POST): Đăng ký khách hàng
- `/api/customers/login/` (POST): Đăng nhập khách hàng
- `/api/customers/<customer_id>/` (GET): Thông tin khách hàng

## Notes
- Quản lý thông tin khách hàng, xác thực, đăng ký, đăng nhập.
